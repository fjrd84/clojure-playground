## Using Leiningen

Let's use [Leiningen](https://leiningen.org/) to run this thing...

- `lein run` Run it!
- `lein uberjar` Build a jar file
- `java -jar target/uberjar/clojure-noob-0.1.0-SNAPSHOT-standalone.jar`

Using the REPL:

- `lein repl` Start it!
